# Checklist de fase 0

Duracion objetivo: 2 semanas

## Semana 1 - Claridad y validacion inicial

- [ ] Elegir zona piloto.
- [ ] Elegir segmento inicial: futbol base, amateur o semiprofesional.
- [ ] Decidir edad minima para beta.
- [ ] Definir si el MVP incluye futbol masculino, femenino o ambos.
- [ ] Listar 20 clubes potenciales para entrevistas.
- [ ] Listar 20 jugadores potenciales para entrevistas.
- [ ] Identificar 2 contactos con conocimiento federativo.
- [ ] Hacer 5 entrevistas a clubes.
- [ ] Hacer 5 entrevistas a jugadores.
- [ ] Documentar problemas repetidos.
- [ ] Confirmar que datos federativos son imprescindibles y cuales son solo deseables.

## Semana 2 - Cierre de alcance

- [ ] Hacer otras 5 entrevistas a clubes.
- [ ] Hacer otras 10 entrevistas a jugadores.
- [ ] Validar flujo de publicacion de busqueda.
- [ ] Validar flujo de postulacion.
- [ ] Validar nivel de confianza requerido para clubes.
- [ ] Validar restricciones de chat.
- [ ] Definir datos minimos del perfil de jugador.
- [ ] Definir datos minimos del perfil de club.
- [ ] Cerrar alcance MVP.
- [ ] Crear backlog inicial de fase 1.
- [ ] Preparar brief para diseno UX/UI.

## Evidencias necesarias

- Notas de entrevistas.
- Lista de clubes interesados.
- Lista de jugadores interesados.
- Decisiones cerradas.
- Riesgos abiertos.
- Alcance MVP ajustado.
- Prioridades de fase 1.

## Criterios para pasar a fase 1

- Hay un segmento piloto claro.
- Hay al menos 10 clubes interesados o contactables.
- Hay al menos 50 jugadores potenciales para captacion inicial.
- El flujo principal club-jugador fue validado.
- La politica sobre menores esta definida.
- La estrategia de datos federativos inicial no bloquea el MVP.
- El alcance MVP no depende de integraciones externas inciertas.

## Primera reunion recomendada

Duracion: 60 minutos.

Agenda:

1. Confirmar vision y objetivo del MVP.
2. Elegir zona piloto.
3. Elegir segmento inicial.
4. Definir politica de edad/minores.
5. Seleccionar contactos para entrevistas.
6. Asignar tareas de semana 1.

## Preguntas que deberiamos responder en la primera reunion

1. Donde tenemos mas acceso real a clubes?
2. Quien seria el primer club piloto ideal?
3. Que jugador representa mejor el usuario inicial?
4. Hay contactos en federaciones territoriales?
5. Queremos construir primero confianza, velocidad de contacto o volumen de oportunidades?
